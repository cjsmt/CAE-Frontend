# 说明
## Vue相关

目前这个框架是用vue3和element-plus搭建的，克隆下来后安装好element等各种组件应该就可以运行。
完成了部分主页设计，路由也配置好了。

目前基本只算个框架，需要把前端的各种功能移植到这上面。

用mock模拟了后端接口，可以获取用于应力场可视化的数据
